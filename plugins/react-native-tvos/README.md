# react-native-tvos Skills

Claude skills for building React Native apps for Apple TV and Android TV.

## Skills

| Skill | Description |
|---|---|
| [tv-project-setup](skills/tv-project-setup/) | Creating new TV projects with Expo or Community CLI |
| [tv-focus-navigation](skills/tv-focus-navigation/) | Focus management with TVFocusGuideView, VirtualizedList, and nextFocus |
| [tv-remote-input](skills/tv-remote-input/) | Handling TV remote and keyboard input |
| [tv-platform-detection](skills/tv-platform-detection/) | Platform detection APIs and TV-specific file extensions |
| [tv-build-configuration](skills/tv-build-configuration/) | Build configuration for New Architecture, Hermes, Podfiles, and Maven |
